# GAIA Template System

## 模块概述

模板系统提供可复用的文档和代码模板，加速项目启动和标准化工作流程。

## 目录结构

```
gaia-templates/
├── engine/          # 模板渲染引擎
├── built-in/        # 内置模板
└── custom/          # 自定义模板
```

## 功能点

| 子模块 | 描述 | 优先级 |
|--------|------|--------|
| 模板引擎 | 参数化模板渲染 | P0 |
| 模板仓库 | 内置和自定义模板存储 | P0 |
| 模板市场 | 社区模板分享 | P2 |
| 继承机制 | 模板组合和继承 | P1 |
| 版本管理 | 模板版本控制 | P1 |

## 模板类型

- PRD 模板 (已有)
- 技术设计模板 (已有)
- 复盘模板 (已有)
- Phase 启动模板 (新增)
- Skill 定义模板 (新增)
